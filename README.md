# MiniPay Balance Monitor

MiniPay Balance Monitor adalah dashboard web real-time untuk memantau saldo treasury MiniPay pada jaringan Celo. Aplikasi ini dibuat untuk menampilkan balance dari beberapa akun on-chain yang digunakan untuk kebutuhan reward dan deposit operasional.

Dashboard ini cocok digunakan untuk monitoring saldo secara cepat, transparan, dan mudah dibaca tanpa perlu membuka explorer satu per satu.

## Fitur Utama

- Monitoring saldo reward MiniPay secara real-time
- Monitoring saldo deposit operasional
- Dukungan token USDC dan USDT
- Menampilkan subtotal per kategori
- Menampilkan status live, block terbaru, dan waktu update terakhir
- Shortcut untuk membuka alamat wallet di Celoscan
- Tampilan dashboard modern, ringan, dan responsif

## Kategori Monitoring

### Rewards

Bagian ini menampilkan saldo akun distribusi reward MiniPay.

Contoh akun yang dimonitor:

- Reward Europe Dev
- Reward US Dev

Informasi yang ditampilkan:

- Nama akun
- Region / jaringan
- Saldo token
- Alamat wallet
- Alamat token
- Link ke Celoscan

### Deposits

Bagian ini menampilkan saldo treasury deposit operasional.

Contoh akun yang dimonitor:

- Deposit USDT
- Deposit USDC

Informasi yang ditampilkan:

- Nama akun deposit
- Token yang digunakan
- Saldo token
- Alamat wallet
- Alamat token
- Link ke Celoscan

## Tech Stack

- Frontend web dashboard
- Celo blockchain
- Celoscan explorer
- USDC / USDT balance tracking
- Real-time block and balance update

## Tujuan Project

Project ini dibuat untuk mempermudah proses pemantauan saldo MiniPay treasury, terutama untuk akun reward dan deposit. Dengan dashboard ini, balance dari beberapa wallet dapat dilihat dalam satu halaman secara real-time.
